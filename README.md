# Jogo de Adivinhação

Fiz esse projeto para treinar lógica de programação depois de uns anos sem programar.

## Como funciona

O sistema gera um número entre 1 e 100 e você tenta adivinhar.  
Ele te diz se o número é maior ou menor até acertar.

## O que pratiquei

- loops (while)
- condições (if/elif)
- tratamento de erro (try/except)
- uso de lista para histórico (JSON)

## Observação

Esse projeto faz parte da minha volta aos estudos de programação.
